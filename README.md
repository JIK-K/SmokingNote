## SmokingNote

<hr/>

### 설명

성공적인 금연과 건강을 위한 운동, 음주, 흡연 기록을 관리하여 건강한 일상을 유지하도록 도움을 주는 건강 앱.

<hr/>

### 메인화면

중앙 화면에 금연을 한 시간을 보여주며, 절약한 금액과 시간, 낭비된 시간과 금액을 보여주는 공간과
3가지의 버튼으로 운동, 음주, 흡연을 기록할 수 있도록 함.
